Bathing water quality data/map viewer - quick help

 Country  - All EU countries are listed, drag the mouse over country name or flag to display country name in native language.

 Region  - Bigger countries are divided into administrative units - regions, select a region to narrow the search.
 "region icon" - Shows a map with all regions in selected country. Click on map to close.

 Province  - Bigger regions are divided into provinces, select a province to narrow the search.
 "province icon" - Smaller countries have only one region (no "region icon"), for these countries a province map is provided. Click on map to close.

 Bathing water  - Initially number of bathing waters in each country are displayed here. When region or province are selected, number of bathing waters in selected region / provice is shown here. Select a province and select box is shown here: select a bathing water and a small window with water quality info will pop up. Brackets indicate status for each year, there are 4 categories/colors. If the bracket is empty (white), there were no measurements or not sufficient samples for that year. Click inside the window to close.

 Visualization 
 - Download and/or open a kml file with bathing water placemarks. If region, province or bathing water are selected, file contains only bathing waters in selected region, province or bathing water. Kml files are best viewed with Google Earth.
